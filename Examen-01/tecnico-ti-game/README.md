# Babylon.js Starter

## Estructura
```
.
├── index.html
├── styles.css
├── game.js
└── assets
    ├── textures
    ├── models
    └── sounds
```

## Servidor local
1. **VS Code Live Server**
   - Instala la extensión "Live Server".
   - Abre `index.html` y pulsa `Go Live`. El sitio se abrirá en `http://127.0.0.1:5500/` o similar.
2. **Alternativa con Node.js**
   - Ejecuta `npx serve .` o `npx http-server .` dentro de la carpeta del proyecto.
   - Entra en la URL que indique la terminal (normalmente `http://localhost:3000`).
